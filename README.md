# Slacky Server

Server-side code: handles Slack authorization process for Slacky

Fork and donwload

Then:

```
// If you want to try it out
yarn start
```

Install the extension in developer mode.

## Usage

* When running, point slacky .env to where server is running
* Enjoy!

* Express
* Metascraper
